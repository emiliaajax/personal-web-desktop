# &lt;my-pwd-icon&gt;

A web component representing a clickable icon.

## Attributes

### `src`

The `src` attribute, if present, specifies the icon image.

Default value: undefined

## Events

| Event Name | Fired When           |
| ---------- | -------------------- |
| `clicked` | The icon is clicked. |

## Example

```html
<my-pwd-icon src=''><my-pwd-icon>
```
