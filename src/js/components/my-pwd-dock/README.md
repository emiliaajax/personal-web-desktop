# &lt;my-pwd-dock&gt;

A web component representing a dock.

## Example

```html
<my-pwd-dock>
    <my-icon src='./images/memory-icon.png'></my-icon>
    <my-icon src='./images/chat-icon.png'></my-icon>
    <my-icon src='./images/snake-icon.png'></my-icon>
</my-pwd-dock>
```
